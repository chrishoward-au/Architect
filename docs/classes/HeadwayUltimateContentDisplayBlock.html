enqueue_action()

enqueue_action( $block_id,  $layout)

Use this to enqueue styles or scripts for your block. This method will be execute when the block type is on the current page you are viewing. Also, not only is it page-specific, the method will execute for every instance of that block type on the current page.

This method will be executed at the WordPress 'wp' hook
